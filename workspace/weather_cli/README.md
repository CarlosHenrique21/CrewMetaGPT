# Weather CLI Tool

## Overview
The Weather CLI Tool is a command-line application designed to fetch and display weather information by querying a third-party weather API.

## Installation
To install the required packages, run:
```
pip install -r requirements.txt
```

## Usage
To get the current weather for a location, use:
```
python cli.py <location>
```
For example:
```
python cli.py London
```

## Running Tests
To run the tests, execute:
```
python -m unittest discover -s tests
```
